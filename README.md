# S & D Lawn Care Website

Static GitHub Pages website for S & D Lawn Care in Ocala, Florida.

## GitHub Pages

1. Push these files to a GitHub repository.
2. In the repository settings, open Pages.
3. Publish from the `main` branch and the root folder.
4. After GitHub Pages gives you the live URL, update the canonical URL, Open Graph URL, JSON-LD URL, `robots.txt`, and `sitemap.xml`.
